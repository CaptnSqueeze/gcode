plot.py

MPCNC pen plotter pipeline  ·  SVG → optimized gcode → Lightburn → machine

Basic usage

python plot.py ~/Downloads/floral.svg floral.gcode a4_portrait --zigzag 0.4 --fill-mode stroke --preview --estimate --mirror-y --compactness 0.05
# minimal — no scaling, no fill python plot.py input.svg output.gcode # scale to A4 landscape python plot.py input.svg output.gcode a4_landscape # scale to A4 portrait python plot.py input.svg output.gcode a4_portrait
Page sizes include a 25mm margin. A4 landscape = 250×190mm, A4 portrait = 190×250mm. Design scales proportionally to fit.

Offset fill

Concentric inward rings — like Lightburn's offset fill. Pen lifts between rings. Looks great for logos and bold shapes.

# offset fill at 0.6mm spacing python plot.py input.svg output.gcode a4_landscape --fill 0.6 # tighter fill for fine detail python plot.py input.svg output.gcode a4_landscape --fill 0.4 # looser fill, faster to plot python plot.py input.svg output.gcode a4_landscape --fill 1.0
Inkscape prep: filled shapes need an actual fill color set (Object → Fill and Stroke → solid fill). Shapes with fill=none are treated as outlines only.

Parallel hatch

Parallel lines swept across filled shapes — classic technical drawing style. Fewer pen lifts than offset fill.

# 45° hatch at 0.6mm spacing (default angle) python plot.py input.svg output.gcode a4_landscape --hatch 0.6 # 30° hatch — shallower angle python plot.py input.svg output.gcode a4_landscape --hatch 0.6 --hatch-angle 30 # crosshatch effect — run twice at 45° and 135° python plot.py input.svg output1.gcode a4_landscape --hatch 0.8 --hatch-angle 45 python plot.py input.svg output2.gcode a4_landscape --hatch 0.8 --hatch-angle 135 # dense technical fill python plot.py input.svg output.gcode a4_landscape --hatch 0.4 --hatch-angle 45 # combine outlines + hatch in one pass python plot.py input.svg output.gcode a4_landscape --hatch 0.6 --fill 0.6

Config (top of plot.py)

MACHINE SETTINGS

QUANTIZATION1.0mm — curve smoothness. Higher = fewer segments, faster. Try 0.5–2.0
MERGE_TOL0.5mm — join path ends within this distance into one stroke
PEN_UP_Z0 — Z position when pen is lifted
PEN_DOWN_Z-80 — Z position when pen touches paper (tune this!)
FEED_Z1000 mm/min — Z axis speed (max per FluidNC config)
FEED_TRAVEL3500 mm/min — XY speed when pen is up
FEED_DRAW3000 mm/min — XY speed when drawing

Workflow

# 1. get image (PNG/JPG/SVG) # 2. if raster: open in Inkscape → Path → Trace Bitmap → save as plain SVG # 3. if you want fill: set fill colors in Inkscape (Object → Fill and Stroke) # 4. run plot.py python plot.py input.svg output.gcode a4_landscape # 5. open output.gcode in Lightburn → send to MPCNC over WiFi

Tips

Ballpoint @ 0.3–0.4mm: hatch spacing of 0.5–0.8mm gives solid fill without overworking the paper. Below 0.4mm and you're just grinding.
Venv: always activate before running — source ~/Documents/mpcnc/plotter/venv/bin/activate
Z tuning: PEN_DOWN_Z = -80 is a starting point for the spring mount. If lines are faint, go more negative. If paper is getting gouged, go less.
